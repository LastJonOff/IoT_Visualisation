#  Запуск окружения

Для начала работы необходимо поднять брокер и базу данных для записи при помощи команды:  
  *sudo docker-compose up -d*
  
После этого запускаем паблишер, находясь в корневом каталоге приложения:  
  *python3 publisher.py*
  
Теперь можем запустить сабскрайбер:  
  *python3 subscriber.py*  
  
При первом запуске появлсяется окно с конфигурацие для сохранения данных в БД, после чего сохраняется файл конфигурации и запись происходит в автоматическом режиме согласно сохраненной конфигурации
